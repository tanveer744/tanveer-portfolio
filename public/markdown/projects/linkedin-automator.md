A Tkinter-based desktop application for automating various LinkedIn tasks using Chrome sessions or manual login. Features include connection automation, post creation, feed engagement, and messaging.

## Features

- **Chrome Session and Credential Login** - Flexible authentication methods
- **Personalized Connection Requests** - Send customized connection invitations
- **AI-based Post Creation** - Generate engaging content automatically
- **Smart Feed Engagement** - Intelligent interaction with posts
- **Custom Message Templates** - Personalized outreach at scale
- **Human-like Behavior Simulation** - Avoid detection with natural patterns
- **Local Data Processing** - Your data stays on your machine
- **Secure Authentication** - Safe credential management
- **Multiple Task Automation** - Run various tasks simultaneously
- **Smart Delay System** - Configurable timing between actions

## Tech Stack

- **Python** - Core programming language
- **Tkinter** - Desktop GUI framework
- **Selenium** - Web automation and browser control
- **ChromeDriver** - Chrome browser automation
- **Gemini API** - AI-powered content generation

## Key Capabilities

### Connection Automation
Send personalized connection requests to targeted profiles with custom notes and smart filtering.

### Post Creation
Leverage AI to generate engaging LinkedIn posts based on your topics and tone preferences.

### Feed Engagement
Automatically like, comment, and engage with posts from your network using intelligent patterns.

### Messaging System
Send bulk messages with personalization variables and template management.

## Safety Features

- Human-like delays and random intervals
- Session persistence to avoid repeated logins
- Rate limiting to prevent account restrictions
- Local data processing for privacy
