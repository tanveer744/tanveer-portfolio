=====================================
WELCOME TO TANVEER'S TERMINAL RESUME
=====================================

This is an interactive terminal-style portfolio showcasing my professional experience, 
projects, skills, and achievements.

BASIC NAVIGATION COMMANDS:
--------------------------
  ls                  → List contents of current directory
  cd <folder>         → Change to a different directory
  cd ..               → Go back to parent directory
  open <file>         → Open and read a file
  clear               → Clear the terminal screen
  help                → Show available commands

QUICK SHORTCUTS:
----------------
  cd about            → Learn about me and my background
  cd projects         → Explore my technical projects
  cd experience       → View my professional experience
  cd education        → Check my educational background
  cd skills           → See my technical and soft skills
  cd certifications   → View my certifications
  cd achievements     → Read about my achievements and publications
  cd interests        → Discover my personal interests

USEFUL COMMAND EXAMPLES:
------------------------
  cd projects; ls                    → Navigate to projects and list them
  cd projects; open hackrx_query_system.txt    → Read about my HackRx project
  cd about; open profile.txt         → View my profile
  cd experience; ls                  → See my work experience

TIPS:
-----
• Use tab completion for faster navigation
• Each .txt file contains detailed information
• Start with 'cd about' to learn more about me
• Check out 'cd projects' to see my best work

Happy exploring!
