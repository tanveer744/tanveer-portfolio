CSS3 (Advanced):
----------------
CSS3 enables modern, responsive, and visually appealing web designs.

CORE SKILLS:
• Modern Layout Systems:
  - Flexbox - One-dimensional layouts, alignment, distribution
  - CSS Grid - Two-dimensional layouts, complex page structures
  - Responsive layouts without media queries (clamp, min, max)

• Styling Techniques:
  - Custom properties (CSS variables) for themeable designs
  - Animations and transitions for smooth interactions
  - Transform and 3D effects
  - Gradient backgrounds and effects
  - Box shadows and visual depth

• Responsive Design:
  - Mobile-first approach
  - Media queries for different breakpoints
  - Fluid typography with clamp()
  - Responsive images and srcset
  - Viewport units (vw, vh, vmin, vmax)

• Advanced Features:
  - CSS preprocessors understanding (Sass/SCSS concepts)
  - BEM methodology for maintainable CSS
  - CSS modules for component scoping
  - Pseudo-elements and pseudo-classes
  - CSS filters and blend modes

PRACTICAL APPLICATIONS:
• Responsive layouts that work across all device sizes
• Beautiful animations and micro-interactions
• Dark mode and theme switching
• Component-based styling architecture
• Performance-optimized CSS

REACT.JS (Proficient):
----------------------
React is the primary framework for building modern, component-based user interfaces.

CORE CONCEPTS:
• Component Architecture:
  - Functional components (modern approach)
  - Class components (legacy understanding)
  - Component composition and reusability
  - Props and prop drilling
  - Children prop for flexible components

• React Hooks:
  - useState - State management in functional components
  - useEffect - Side effects and lifecycle
  - useContext - Context API for state sharing
  - useRef - DOM references and mutable values
  - Custom hooks - Reusable stateful logic
  - useMemo & useCallback - Performance optimization

• State Management:
  - Local component state
  - Context API for global state
  - State lifting patterns
  - Redux basics (if needed)
  - State immutability principles

• React Patterns:
  - Controlled vs Uncontrolled components
  - Render props pattern
  - Higher-Order Components (HOC)
  - Compound components
  - Conditional rendering techniques

REACT ECOSYSTEM:
• React Router - Client-side routing and navigation
• Form handling - Controlled inputs, validation
• HTTP requests - Axios/Fetch integration
• Error boundaries - Graceful error handling
• Code splitting - Dynamic imports for performance

PRACTICAL APPLICATIONS:
• Single Page Applications (SPAs)
• Dynamic user interfaces with real-time updates
• Reusable component libraries
• Complex form handling with validation
• Interactive dashboards and data visualization
