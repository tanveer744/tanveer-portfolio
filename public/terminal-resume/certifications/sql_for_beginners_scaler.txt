================================================================================
COURSE OVERVIEW
================================================================================

This certification from Scaler provides comprehensive training in SQL (Structured
Query Language), the standard language for managing and manipulating relational
databases. The course covers fundamental to intermediate SQL concepts essential
for software developers, data analysts, and backend engineers.

SQL is a critical skill for any developer working with data-driven applications,
and this certification represents solid foundational knowledge in database
querying and management.

================================================================================
TOPICS COVERED
================================================================================

INTRODUCTION TO DATABASES:
--------------------------
• What is a database and DBMS
• Relational database concepts
• Tables, rows, and columns
• Primary keys and foreign keys
• Database normalization basics
• Entity-Relationship (ER) diagrams

BASIC SQL QUERIES:
------------------
• SELECT statement for data retrieval
• WHERE clause for filtering data
• Selecting specific columns
• Using DISTINCT for unique values
• LIMIT and OFFSET for pagination
• Sorting with ORDER BY (ASC, DESC)

FILTERING AND OPERATORS:
------------------------
• Comparison operators (=, !=, >, <, >=, <=)
• Logical operators (AND, OR, NOT)
• BETWEEN for range queries
• IN operator for multiple values
• LIKE for pattern matching
• IS NULL and IS NOT NULL
• Combining multiple conditions

AGGREGATE FUNCTIONS:
--------------------
• COUNT for counting rows
• SUM for totaling values
• AVG for calculating averages
• MAX and MIN for extremes
• GROUP BY for aggregation grouping
• HAVING clause for filtered aggregations
• Using aggregate functions effectively

JOINS:
------
• Understanding table relationships
• INNER JOIN for matching records
• LEFT JOIN (LEFT OUTER JOIN)
• RIGHT JOIN (RIGHT OUTER JOIN)
• FULL JOIN for all records
• CROSS JOIN for Cartesian products
• Self joins for hierarchical data
• Multiple table joins

SUBQUERIES:
-----------
• Nested SELECT statements
• Subqueries in WHERE clause
• Subqueries in FROM clause (derived tables)
• Correlated subqueries
• EXISTS and NOT EXISTS
• IN with subqueries
• Scalar subqueries

DATA MANIPULATION:
------------------
• INSERT for adding new records
• UPDATE for modifying existing data
• DELETE for removing records
• TRUNCATE vs DELETE
• Data integrity considerations
• Transaction basics (COMMIT, ROLLBACK)

ADVANCED CONCEPTS:
------------------
• String functions (CONCAT, SUBSTRING, LENGTH, UPPER, LOWER)
• Date and time functions
• CASE statements for conditional logic
• COALESCE for NULL handling
• Type conversion (CAST, CONVERT)
• Window functions basics
• Common Table Expressions (CTEs) introduction

DATABASE DESIGN:
----------------
• Creating tables with CREATE TABLE
• Defining data types (INT, VARCHAR, DATE, etc.)
• Primary key constraints
• Foreign key constraints
• UNIQUE and NOT NULL constraints
• DEFAULT values
• Basic indexing concepts
